# Tile_Layout_Calculator_GH

タイルの割付けと目地を計算するGrasshopper定義です。

## 動作環境
- Rhinoceros 8
- Grasshopper

## 使い方
1. Rhinoでカーブを作成
2. .ghファイルを開く
3. スライダーでパラメータを調整

## パラメータ
- タイルサイズ：タイルの寸法調整
- 目地寸法：タイル間の目地幅の調整
- タイル枚数：指定範囲内の割付け枚数
