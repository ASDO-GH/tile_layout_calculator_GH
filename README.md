tile_layout_calculator_GH
Parametric tile layout and joint calculator for Grasshopper.

タイルの割付けと目地を計算するGrasshopper定義です。

動作環境
Rhinoceros 8

Grasshopper

使い方
Rhino上でベースとなるカーブ（またはサーフェス）を作成。

本GHファイルを開き、必要なジオメトリをセット。

スライダーを動かして各パラメータを調整してください。

パラメータ
タイルサイズ: 各タイルの寸法調整

目地寸法: タイル間の目地幅の調整

タイル枚数: 指定範囲内の割付け枚数のシミュレーション
